Le club : Union Sportive Soudanaise

par Thomas Garnache

Bonjour à vous qui visitez ce site !

Vous vous trouvez sur le site du club de football de la commune de Soudan : l'Union Sportive Soudanaise (USS). C'est un club très actif dans lequel on retrouve les jeunes joueurs, les joueurs séniors et toutes les personnes qui participent de près ou de loin à sa vie active : le personnel du Bureau, les arbitres et tous les bénévoles qui sont d'une aide précieuse. Le club possède une école de football qui lui permet d'avoir des joueurs dans les catégories les plus jeunes : de U6 jusqu'à U13 ! Les joueurs U15 à U18 se retrouvent dans un groupement jeune avec les communes voisines.

Les entraînements séniors se déroulent tous les mardi soirs et vendredi soirs à partir de 19H15.
